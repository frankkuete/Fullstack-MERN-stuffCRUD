# Fullstack-MERN-stuffCRUD

## Run the backend server
- cd backend 
- npm install 
- nodemon server

## Run the Forntend 
- cd frontend
- npm install 
- npm start 

## Configure the MongoDB ATLAS CONNECTION URI 
- cd backend 
- create .env file 
- add the line => ATLAS_URI = mongodb-atlas-connection-uri (without quotes)
- that's your connection is ok
